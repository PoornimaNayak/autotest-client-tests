This package contains the Linux user space daemon and configuration tool for
Intel LLDP Agent with Enhanced Ethernet support for the Data Center

This package covers testing of the below binaries with several options 
/usr/sbin/lldpad
/usr/sbin/dcbtool
/usr/sbin/lldptool

lldptool and dcbtool binaries can be tested by enabling the lldpad daemon (lldpad -d)

HOW TO RUN:
cd /opt/fiv/ltp/testcases/fivextra/lldpad
./lldpad.sh
